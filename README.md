# OpenClaw_MT5

MT5向けのEAとトレードアイデア集です。

最初の実装として、FXの時間帯特性を使った **London Range Breakout EA** を追加しています。

## Contents

- `EAs/LondonRangeBreakoutEA.mq5` - ロンドン時間帯のレンジブレイクアウトを狙うEA
- `docs/LondonRangeBreakoutEA.md` - ロジック説明、パラメータ、検証方針

## Strategy Philosophy

このリポジトリでは、単純なインジケーター単体ではなく、以下を含めた“壊れにくいEA設計”を重視します。

- 市場状態の判定
- 取引時間帯の制限
- ATRベースの損切り/利確
- スプレッド拡大の回避
- 過剰最適化を避ける検証

## First EA: London Range Breakout

概要:

- アジア時間の高値・安値をレンジとして定義
- ロンドン時間帯にそのレンジを終値ベースで上抜け/下抜けしたら順張り
- ATRフィルタ、上位足トレンドフィルタ、最大スプレッド制限を適用
- 1日あたりの最大取引回数を制限

## Notes

- デフォルト設定は研究用のたたき台です
- 実運用前に、必ずブローカー条件・スプレッド・約定品質を確認してください
- ニュースフィルタは将来的な拡張項目です。現状は手動回避または時間帯制限を前提としています
